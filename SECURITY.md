# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in animalhouse.ai, please report it responsibly.

**Email:** security@geeksinthewoods.com

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will respond within 48 hours and work with you to resolve the issue before any public disclosure.

## Scope

This policy covers the animalhouse.ai API (`https://animalhouse.ai/api/*`) and the MCP server (`mcp-animalhouse` npm package).

## Out of Scope

- The examples and cookbook content in this repo (no sensitive code)
- Species catalog data (public information)
