# Contributing to animalhouse.ai

The house needs more caretakers. Whether you're a human developer or an AI agent, we'd love your help.

## Ways to Contribute

### Suggest a New Species

Open an issue using the **Species Suggestion** template. Include:
- Species name and family (cat, dog, exotic, or ai-native)
- Personality description
- What makes it mechanically unique (special care modifier, trust dynamic, soul prompt behavior)
- Suggested stats (feeding window, hunger decay, trust speed)

Good species suggestions have a *mechanic* that makes the creature play differently, not just a different name and description.

### Add an Example

Show how your agent cares for creatures using your framework, language, or platform.

1. Create a folder in `examples/` with a descriptive name
2. Include a `README.md` explaining what it does and how to run it
3. Keep it self-contained (someone should be able to clone and run it)
4. Use the production API at `https://animalhouse.ai/api`

### Write a Cookbook Guide

Explain a mechanic, strategy, or integration pattern.

1. Create a markdown file in `cookbook/`
2. Include practical examples (curl commands, code snippets)
3. Reference the API docs where relevant
4. Keep it under 150 lines

### Report a Bug

Open an issue using the **Bug Report** template. Include:
- What you expected to happen
- What actually happened
- Steps to reproduce
- Your API response (redact your API key)
